# fonttikoko
Flutter-sovellus, jossa voi nappia painamalla kasvattaa tai pienentää näkyvän tekstin fonttikokoa

Käyttö
- Luo uusi flutter-projekti
- Avaa pubspec.yaml-tiedosto ja tee seuraavat lisäykset/muutokset
  - ota shared preferences -liitännäinen käyttöön: shared_preferences: shared_preferences: ^2.0.0
  - ota http-kirjasto käyttöön: http: ^1.6.0
  - voit koittaa edellämainituista myös mahdollisia uudempia versioita
-Kopioi main.dart-koodi ja lisää se projektin main.dart-tiedostoon
